# 「버튼 잡기 게임」 (C# 코딩)

## ★개요
- C# 프로그래밍 학습
- 핵심기능: 1단계 버튼 이동(마우스가 버튼 위로 이동하면 버튼의 위치 변경),
  2단계 효과음과 메시지 박스(버튼이 이동할 때마다 효과음 발생, 버튼을 잡았을 때 효과음 발생과 축하합니다 메시지 박스 제시)
  3단계 점수판 추가와 버튼 크기 축소(폼 제목에 점수판 추가, 버튼을 잡으면 +100점/놓치면 -10점, 버튼 잡기에 성공할 때마다 버튼의 크기를 10%씩 축소)
  4단계 게임오버와 재시작(버튼을 20번 놓치면 게임오버됨, 게임오버 후 처음 화면과 같이 점수 등이 초기화되며 재시작됨)
- 화면구성: 흰 바탕에 버튼 하나가 있음.
  폼 제목에는 게임 이름, 버튼의 위치(x,y)와 점수가 표시됨.

## ★구현 시 어려웠던 점 (구현 과정)
- 난수생성기를 이용해 버튼의 X좌표와 Y좌표가 랜덤으로 바뀌도록 해 버튼이 도망가는 모습을 구현함. (화면이 폼 밖으로 이동하지 않도록 할 필요가 있었음)
- 버튼을 잡기에 성공했을 때 MessageBox.Show를 사용해 "축하합니다~!"라는 메시지 박스가 나타나도록 구현함.
- System.Media를 사용해 버튼을 놓쳤을 때와 잡았을 때 각각 효과음이 나도록 구현함. (직접 사운드 파일을 삽입하는 대신 .NET에 있는 기본 시스템 사운드를 재생하도록 함 -> 사운드 파일을 직접 가져와야 하는 줄 알았으나 이런 방법이 있었음)
- GameOver()와 ResetGame()이라는 함수를 만들어 게임 오버와 재시작을 구현함. (GameOver()가 실행된 후 바로 ResetGame()이 실행되도록 함)
